# Calculadora 
Este proyecto consiste en una calculadora desarrollada en Python que incluye operaciones matemáticas básicas, conversiones de unidades de datos y conversiones entre sistemas numéricos.

El objetivo del proyecto es aplicar conceptos fundamentales de programación en Python como funciones, condicionales, listas, manejo de archivos.

## Integrantes

- German Jimenez Garcia — Desarrollo de calculadora,Sistema de historial y exportación,Conversión de unidades

## Características

- Calculadora básica
- Operaciones: suma, resta, multiplicación, división, módulo y potencia
- Conversión de unidades:
- Bytes
- Kilobytes
- Megabytes
- Gigabytes
- Conversión de sistemas numéricos:
- Decimal a Binario
- Decimal a Hexadecimal
- Binario a Decimal
- Historial de operaciones
- Exportación de historial a CSV
- Interfaz gráfica usando Tkinter
- Aplicación ejecutable para Windows


## Tecnologías utilizadas

- Python 3.x
- Tkinter
- JSON
- CSV

## Instalación

1. Clonar el repositorio
2. Entrar a la carpeta
3. Ejecutar el programa


## Uso

1. Introducir números usando los botones
2. Seleccionar la operación
3. Presionar "=" para obtener el resultado
4. Utilizar las herramientas de conversión disponibles
